## 运行
> 在运行前，请保证在你的电脑上安装了[nodejs](https://nodejs.org/en/)(我使用的版本是v16.14.0), [git](https://git-scm.com/)

项目运行步骤：
```bash
git clone https://github.com/zone-1614/uml.git
cd uml
npm install
npm run serve
```
然后在本机的浏览器上访问8082端口，即可看到界面
